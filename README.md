# ZRH·5 — Zürich Reality Check

Single-file survival calculator for entering the Zürich ride-hailing market with a family of five (2026 conditions: Uber–Bolt price war, 20–25% commissions, AHV/SUVA + tax, CH vehicle costs).

- Survival tachometer: required km/day vs. physical/legal capacity
- 15 sliders: household, market, vehicle, regime
- Plan B: vehicle strategy presets (Standard / Comfort mix / Lux-Black), private fixed-price transfers, car second-shift rent-out, IT side income
- Live scenario comparison + sensitivity levers
- BG/EN

Deploy: GitHub Pages → root. No build, no dependencies.
